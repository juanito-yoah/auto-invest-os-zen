# auto-invest-os-zen

auto-invest-os-zen is an AI-powered investment research platform that combines real-time market data from Yahoo Finance with Claude AI analysis to surface actionable investment insights. It uses a SQLite database to persist historical data and research results, data pipelines to fetch and process market information, and a Flask REST API to expose research endpoints — enabling users to query, analyze, and reason over equities with the power of a large language model.
