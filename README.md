# auto-invest-os-zen

AI-powered investment research platform with automated SEC filing analysis, portfolio tracking, and intelligent insights. Built with Python, Claude API, and React. Note: this is for personal use/portfolio demos, so not offering financial advice here to others : )

auto-invest-os-zen combines real-time market data from Yahoo Finance with Claude AI analysis to surface actionable investment insights. It uses a SQLite database to persist historical data and research results, data pipelines to fetch and process market information, and a Flask REST API to expose research endpoints — enabling users to query, analyze, and reason over equities with the power of a large language model.
